Sistema de Recomendaciones en Red Social 

Este proyecto en C++ implementa un sistema que sugiere intereses a los usuarios basándose en lo que les gusta a sus amigos. La similitud se calcula con el índice de Jaccard y las recomendaciones se generan acumulando los intereses de los amigos más parecidos.
